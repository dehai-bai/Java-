# 内部类
### 是什么
内部类就是将一个类定义在另一个类中；

### 为什么用
- 可以增强两个类之间的练习，并可以使程序代码清晰，简洁；
- 优点：对只在一处使用的类进行分组；提高封装性；增强代码的可读性和可维护性；

### 怎么用
按照内部类是否使用static修饰，可分为两类：静态的与非静态的；其中非静态的包括成员内部类，局部内部类，匿名内部类和静态内部类。
